隐私政策

隐私政策

1. 简介

FlashLingo 是一款用抽认卡、间隔重复、音频、本地卡组、统计和按卡组手动备份来学习词汇的 Android 应用。本政策说明哪些信息被本地存储、哪些第三方服务可能处理数据，以及你拥有哪些选择。

2. 快速摘要

  • 无需在线账户。FlashLingo 不索取你的姓名、电子邮箱、密码、头像、电话号码或支付信息。
  • FlashLingo 不运营自有后端服务器、云同步、自有分析或自有的崩溃报告。
  • 导入的卡组、学习进度、设置、会话、复习历史、统计和本地媒体，存储在你设备上应用的私有存储区域。
  • 应用通过 Google AdMob / Google Mobile Ads SDK 显示广告。Google 可能依据其自身政策处理设备、广告、诊断、大致位置和广告互动数据。
  • 应用数据的 Android 系统自动备份和设备间传输在发布配置中被禁用。你可以用按卡组的 .flashjp 备份手动迁移进度。

3. 本地存储的数据

由于 FlashLingo 不运营自有应用后端，以下数据被本地存储，不会发送到 FlashLingo 的服务器：

  • 导入的卡组及其内容，包括单词、含义、句子、读音、形式、翻译、图片、音频和卡组图标。
  • 学习进度，包括卡片状态、间隔重复排程、复习日期、正确和错误的回答、学习步骤、失误、会话进度和学习时间。
  • 统计，包括每日统计、复习历史、最近会话、问题卡片指标和预测信息。
  • 应用和卡组设置，例如语言、主题、引导状态、每日上限、提醒、书写模式、撤销和间隔重复参数。

4. 广告与 Google AdMob

FlashLingo 使用 Google AdMob / Google Mobile Ads SDK 显示广告。FlashLingo 不会将你的姓名、邮箱、卡组内容、卡片回答、导入的文件或学习进度添加到广告请求中。

根据 Google 的文档，Google Mobile Ads SDK 可能收集和处理诸如 IP 地址、广告标识符、设备或其他标识符、应用互动、广告互动、诊断信息、崩溃或性能信息，以及从 IP 地址推断的大致位置等数据。Google 可能将这些数据用于广告、广告测量、分析、欺诈防范、安全和服务运营。

Google 资源：Google 隐私政策和AdMob 隐私文档。

5. 同意与隐私选择

在必要时，包括针对欧洲经济区、英国和瑞士的用户，FlashLingo 使用 Google 用户消息平台（UMP）在加载广告前请求或刷新同意。仅当应用在当前同意状态下被允许请求广告时才会请求广告。如果 Google Mobile Ads 报告你所在地区需要隐私选项，FlashLingo 可能在应用设置中显示一个入口，供你重新打开广告隐私选择。

你也可以通过 Android 或你的 Google 账户设置管理广告选项，包括在受支持时重置或限制广告 ID。

6. 权限

  • 互联网与网络状态：供 Google Mobile Ads 及需要了解连接情况的平台组件使用。
  • 广告 ID 与 Android AdServices 权限：供 Google Mobile Ads 用于广告、测量、归因，以及在可用时的隐私沙盒功能。
  • 通知：仅在你启用并在必要时授予 Android 通知权限后，用于本地学习提醒。
  • Receive boot completed：用于在设备重启后重新安排已启用的本地学习提醒。

FlashLingo 不请求相机、麦克风、精确 GPS 位置、通讯录、日历、短信、电话、广泛存储、照片、视频或音频库权限。

7. 导入、导出与分享

FlashLingo 允许你通过 Android 文件选择器导入官方 .flashjp 卡组和应用生成的 .flashjp 进度备份。应用会将所选文件复制到其自有的临时或私有存储中进行处理。

你可以将统计导出为 CSV/PDF，将按卡组的进度备份导出为 .flashjp 文件。仅在你选择导出操作时才会创建导出，仅在你通过 Android 分享面板选择目标时才会分享。导出的文件可能包含卡片文本、本地媒体引用、复习历史、会话、设置、统计和学习指标。在版本 1 中，用户进度备份没有密码保护或用户加密。

8. 保留与删除

本地应用数据会保留在你的设备上，直到你通过卸载 FlashLingo、在 Android 设置中清除应用数据、在应用内删除卡组，或使用任何可用的删除/重置功能将其删除。当你删除卡组时，FlashLingo 会依据应用的删除行为，删除该卡组的卡片、设置、日志、会话、统计和孤立的本地媒体。

对于由 Google AdMob 处理的数据，请参阅 Google 的隐私控制和政策。

9. 安全

FlashLingo 将应用数据存储在操作系统的私有应用存储中。官方卡组在导入前会经过验证。导入的卡片 HTML 在学习 WebView 中渲染前会被净化，导入 HTML 中的远程资源会依据应用的安全模型被拦截。没有任何存储或传输方法是绝对安全的；请妥善保管你的设备、导出的文件和备份。

10. 儿童

FlashLingo 不面向 13 岁以下的儿童。由于应用使用可能涉及广告标识符的广告技术，FlashLingo 面向 13 岁及以上的用户。

11. 你的权利

根据你的管辖区，你可能拥有访问、更正、删除、限制、可携带、反对以及在处理基于同意时撤回同意的权利。由于 FlashLingo 不将你的应用学习数据保存在自有服务器上，关于应用数据的大多数权利都可以通过删除卡组、清除应用数据、卸载应用或更改应用设置，直接在你的设备上行使。

12. 第三方链接

应用和网站可能链接到外部网站或服务，包括 Google、Google Play、GitHub Pages、GitHub Releases 或用于官方下载的其他文件托管方。这些第三方拥有各自的隐私政策和条款。

13. 变更与联系

我们可能会不时更新本隐私政策。现行版本将连同最后更新日期发布在本页面上。如有隐私问题，请联系 flashlingo.lenguageapp@gmail.com。
